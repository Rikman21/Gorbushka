## 🗄 Новая структура базы данных

### Таблицы:

1. **users** - пользователи с ролями и рейтингом
2. **catalog** - каталог товаров (управляется админом)
3. **offers** - предложения от поставщиков
4. **deals** - сделки между сторонами
5. **messages** - чат внутри сделки
6. **reviews** - отзывы о поставщиках

### Миграция:

Старая БД не совместима. Нужно:
1. Удалить старый `market.db`
2. Запустить `python test_data.py`
3. Или создать вручную: `python -c "import database; database.init_db()"`

## 🔌 Новые API endpoints

- `GET /api/offers` - список предложений с фильтрами
- `GET /api/catalog` - каталог товаров
- `GET /api/user` - данные пользователя
- `GET /api/deals` - сделки пользователя
- `GET /api/deal` - детали сделки с чатом
- `GET /api/supplier` - профиль поставщика

Старые endpoints (`/api/offers` для Excel) удалены.
