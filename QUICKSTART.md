# Быстрый запуск Биржи Горбушка

## За 5 минут

### 1. Установите зависимости

```bash
pip install -r requirements.txt
```

### 2. Добавьте тестовые данные

```bash
python test_data.py
```

Это создаст:
- ✅ 12 товаров в каталоге
- ✅ 2 поставщиков с предложениями
- ✅ 8 предложений на бирже

### 3. Настройте index.html

Откройте `index.html` и найдите строку:
```javascript
const API_BASE = urlParams.get('api') || 'https://your-server.com';
```

**Вариант А: Локальное тестирование через ngrok**

```bash
# В одном терминале
python main.py

# В другом терминале
ngrok http 8080
```

Скопируйте URL (например: `https://abc123.ngrok-free.app`) и замените в `index.html`:
```javascript
const API_BASE = urlParams.get('api') || 'https://abc123.ngrok-free.app';
```

**Вариант Б: Использование сервера**

Замените на адрес вашего сервера:
```javascript
const API_BASE = urlParams.get('api') || 'https://gorbushka.example.com';
```

### 4. Загрузите index.html на GitHub Pages

```bash
git add index.html
git commit -m "Update API endpoint"
git push
```

Подождите 1-2 минуты пока GitHub Pages обновится.

### 5. Запустите бота

```bash
python main.py
```

Вы увидите:
```
INFO:root:База данных инициализирована
INFO:aiohttp.access:API сервер запущен на порту 8080
INFO:aiogram:Бот запущен
```

### 6. Проверьте API

Откройте в браузере:
```
http://127.0.0.1:8080/api/offers
```

Должен вернуться JSON с предложениями.

### 7. Откройте бота в Telegram

1. Напишите `/start` вашему боту
2. Нажмите "📱 ОТКРЫТЬ БИРЖУ"
3. Вы увидите список предложений!

## Решение проблем

### Бот не отвечает
- Проверьте, что `python main.py` запущен
- Проверьте TOKEN в `main.py`

### WebApp показывает вечную загрузку
- Проверьте, что API доступен: `curl http://127.0.0.1:8080/`
- Проверьте `API_BASE` в `index.html`
- Проверьте, что GitHub Pages обновился (может занять 10 минут)

### Предложения не загружаются
- Проверьте консоль браузера (F12)
- Проверьте CORS: API должен возвращать заголовок `Access-Control-Allow-Origin: *`
- Запустите `python test_data.py` еще раз

### ngrok выдает ошибку
- Зарегистрируйтесь на https://ngrok.com
- Получите токен и выполните: `ngrok config add-authtoken YOUR_TOKEN`
